Packages for telegram portion
pip install telegram
pip install python-telegram-bot


Telegram.csv structure is as follows
key,chat id
API telegram_bot_key:API hash,-mychatidwithdash

To get chat id, go to telegram web, open chat channel, copy the browser link(e.g. https://web.telegram.org/#/im?p=g123456789), take last number from url and convert 'g' to dash '-' to form the chat ID which should look something like the example below.

Example:
key,chat id
123456789:sdnvdfnvnsv032i29j32nkjrnkjef34r,-123456789
