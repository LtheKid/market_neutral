Packages for telegram portion
pip install telegram
pip install python-telegram-bot


Telegram.csv structure is as follows
key,chat id
API telegram_bot_key:API hash,-mychatidwithdash

To get chat id, go to telegram web, open chat channel, take last number from url and add a dash before it '-'

Example:
key,chat id
123456789:sdnvdfnvnsv032i29j32nkjrnkjef34r:-12345
